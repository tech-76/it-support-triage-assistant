# Service Overview

## Project name

IT Support Triage Assistant

## One-line description

A service-desk intake tool that turns IT support requests into structured troubleshooting reports.

## Purpose

The IT Support Triage Assistant is designed to improve how support requests are collected, organized, prioritized, and documented before a technician begins hands-on troubleshooting.

Instead of receiving unclear messages such as “my email is not working,” the tool helps convert the request into a structured support summary with category, priority, escalation guidance, troubleshooting steps, technician notes, and a user-friendly response.

## Business problem

Small businesses and support teams often receive incomplete IT support requests. Missing details create delays, extra back-and-forth, unclear priority, and inconsistent documentation.

This project helps reduce that problem by creating a repeatable intake process.

## What the assistant produces

For each ticket, the assistant generates:

- Ticket summary
- Category
- Priority
- Escalation recommendation
- User impact
- Likely causes
- Troubleshooting steps
- Technician notes
- User-friendly response
- Recommended next action
- Confidence score
- Downloadable report

## Intended users

- Small businesses that need cleaner IT support intake
- Freelance IT support technicians
- Help desk and service desk teams
- Portfolio visitors and recruiters
- Technical support learners practicing ticket documentation

## Current version

Version: v2.1

The current version focuses on structured triage logic, professional documentation, and a working Hugging Face Spaces deployment.

## Important privacy note

The tool should not be used to collect passwords, MFA codes, recovery codes, private keys, payment information, confidential business information, or customer personal data.

## Positioning

This is not positioned as a general chatbot. It is positioned as a support workflow tool.

Strong positioning:

> A service-desk intake assistant that turns messy IT support requests into structured troubleshooting reports.

Avoid weak positioning:

> An AI chatbot for IT.

## Potential service use

This project could support a future small-business IT service offering by helping organize incoming requests, document issues, and prepare technician-ready handoff notes.

Potential future service models include:

- Free public demo
- Custom setup for small businesses
- Internal tool for freelance support work
- Monthly support workflow add-on
- White-label version for small teams
